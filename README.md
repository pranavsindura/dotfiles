## Usage

```bash
cd $HOME/.config
ln -s ~/SourceCode/dotfiles/lazygit lazygit
ln -s ~/SourceCode/dotfiles/ranger ranger
ln -s ~/SourceCode/dotfiles/kitty kitty
ln -s ~/SourceCode/dotfiles/alacritty alacritty
ln -s ~/SourceCode/dotfiles/tmux tmux
```
